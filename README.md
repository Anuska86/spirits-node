👻 The Spirits Node
A full-stack, atmospheric web application for recording and exploring paranormal sightings. Built with a "Modern-Occult" aesthetic, this project focuses on high-performance asset delivery and immersive user experience.

🚀 Quick Start
Clone the repository:

Bash
git clone https://github.com/Anuska86/spirits-node.git
Install dependencies:

Bash
npm install
Run the server:

Bash
npm start
Visit the portal:
Go to http://localhost:8000/

🛠️ Tech Stack & Features:

Backend: Vanilla Node.js

- Custom Routing: Built a lightweight server from scratch to handle static file serving and API requests without heavy frameworks.

- MIME Management: Implemented a robust getContentType system to serve diverse assets like .ico, .mp3, .json, and .png with correct headers.

Frontend: Modern Spooky UI:

- Atmospheric CSS: Utilizes Glassmorphism, custom background overlays, and CSS Grid/Flexbox for a responsive, "found-footage" feel.

- Responsive Layouts: Designed with clamp() typography and flexible containers to ensure investigators can log sightings from any device.

- Interactive Elements: Features a persistent audio atmosphere toggle and dynamic card expansion for reading full reports.

📂 Project Structure:

- public/: All static assets including the custom favicon, images, and atmospheric audio.

- server.js: The core Node.js logic for the application.

- styles/: Modular CSS files (Shared, Sightings, News, Upload) for easy maintenance.
